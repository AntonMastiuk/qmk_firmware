# Lily58
### Left encoder
##### QWERTY (0)
- Volume Up/Down
##### RAISE (2)
- Previous/Next Track
### Right encoder
##### QWERTY (0)
- Navigate Back and Forward
##### RAISE (2)
- CTRL + Left / CTRL + Right
##### ADJUST (3)
- Increase / Decrease brightness of the RGB light


![](https://i.imgur.com/tEOZVRo.png)
Flashing example for this keyboard:

    qmk flash -kb lily58/glow_enc -km 0ffton_rev2
